## Getting Started

1. **Fork** this repository on GitHub
2. **Clone** your fork: `git clone https://github.com/[your-username]/dev-toolbox.git`
3. **Install**: `npm install`
4. **Start**: `npm run dev` 
5. **Have your AI explore** the project structure
